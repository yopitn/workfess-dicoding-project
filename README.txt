Disini saya menggunakan JavaScript pada beberapa element.
Pertama pada Hamburger Menu, Navbar menu link;

Saya menarapkan event click pada Hamburger Menu, dan saat user mengklik
Hamburger Menu pada tampilan Mobile, Navbar Menu akan muncul dari sebelah
kanan.

Kemudian, saya menerapkan fungsi pada Navbar Link. Dimana, Navbar Link akan
berubah saat user meng-scroll halaman. Dimana akan ada perbedaan warna 
antara Navbar Link yang active dan tidak.

Lalu, saya juga menambahkan event pada Navbar Link. Dimana, jika Navbar
Link tersebut user klik pada tampilan Mobile. Maka akan menutup Navbar Menu.
